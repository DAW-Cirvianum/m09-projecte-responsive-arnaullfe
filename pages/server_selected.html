Servidor PHP

Informació

Un servidor web és un programari de servidor, o maquinari dedicat a executar aquest programari, que pot satisfer les sol·licituds dels clients a la World Wide Web. Un servidor web pot, en general, contenir un o més llocs web. Un servidor web processa les sol·licituds de xarxa entrants mitjançant HTTP i diversos protocols relacionats.
